## Usage

Use Title component to render h1-h6 headings with CoengageUI theme styles.
By default, Title has no margins and paddings.
You can change headings font-size and line-height properties in [theme](/theming/extend-theme/).

Pass `order` prop to render specific element (h1-h6), default order is 1:
