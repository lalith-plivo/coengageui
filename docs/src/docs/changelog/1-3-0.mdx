## Other changes

- [New documentation design](https://coengage.dev) with tabs and less cluttered packages index pages
- All packages are now published in 3 formats: esm (ECMAScript modules `export { Button } from './Button';`), cjs (commonjs `export.Button = Button;`) and umd (default for browser). These changes will allow webpack and similar bundlers to perform better tree shacking.
- All components now export related types: `import { ButtonProps } from '@coengageui/core';`
- [use-focus-trap](https://coengage.dev/hooks/use-focus-trap/#initial-focus) hook now allows to change element that will initially receive focus

## Design changes

- [Button](https://coengage.dev/core/button/) component label is no longer uppercased by default, to replicate old design use `<Button uppercase />`
- Colors saturation and text shadows adjusted for [Button](https://coengage.dev/core/button/) and [Badge](https://coengage.dev/core/badge/) components for better contrast
