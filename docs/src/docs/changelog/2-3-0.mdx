## New dark theme

Dark theme colors were adjusted in all components to have better contrast, overall all `theme.colors.dark` colors are darker now.

## Popper.js integration

Components that use any kind of overlay are now powered by [popper.js](https://popper.js.org/). Full list of components that were migrated:

- [Menu](https://coengage.dev/core/menu/)
- [Tooltip](https://coengage.dev/core/tooltip/)
- [Popover](https://coengage.dev/core/popover/)
- [DatePicker](https://coengage.dev/dates/date-picker/)
- [DateRangePicker](https://coengage.dev/dates/date-picker/)

## Other changes

- [Group](https://coengage.dev/core/group/) component now support `withGutter` prop to add spacing to all sides of group
- [use-click-outside](https://coengage.dev/hooks/use-click-outside/) now has an API to handle multiple nodes
- [Transition](https://coengage.dev/core/transition/) component was migrated to native implementation – it does not depend on react-transition-group any more (#93)
- [ColorSwatch](https://coengage.dev/core/color-swatch/) component now supports transparent colors and has better contrast for light colors
- [GlobalStyles](https://coengage.dev/theming/global-styles/) component now sets `box-sizing: border-box` on all elements
- Dropdown styles of [Select](https://coengage.dev/core/select/), [MultiSelect](https://coengage.dev/core/multi-select/) and [Autocomplete](https://coengage.dev/core/autocomplete/) components were adjusted to match other components design
- [Slider](https://coengage.dev/core/slider/) component has new thumb styles
